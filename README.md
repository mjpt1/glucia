# گلوسیا — پلتفرم جامع مدیریت دیابت

پلتفرم حرفه‌ای مدیریت دیابت با رابط کاربری فارسی RTL، هوش مصنوعی، و داشبوردهای پزشکی پیشرفته.

## معماری

```
glucia/
├── apps/
│   ├── api/          # NestJS + Prisma + PostgreSQL
│   └── web/          # Next.js 15 + TailwindCSS + Framer Motion
└── docker-compose.yml
```

## راه‌اندازی سریع

### پیش‌نیازها
- Node.js 20+
- Docker & Docker Compose
- PostgreSQL

### اجرا با Docker
```bash
docker compose up -d
```

### اجرا محلی
```bash
# 1. نصب وابستگی‌ها
npm install

# 2. کپی فایل محیطی
cp apps/api/.env.example apps/api/.env

# 3. ساخت پایگاه داده
cd apps/api && npx prisma migrate dev && npm run db:seed && cd ../..

# 4. اجرا
npm run dev
```

## ویژگی‌ها
- پنل بیمار: ثبت قند خون، وعده‌های غذایی، نوبت‌ها، گزارش‌ها
- پنل پزشک: مدیریت بیماران، نسخه‌نویسی، نوبت‌ها
- پنل مدیر: مدیریت کاربران، گزارش‌های سیستم
- کوچ هوشمند: چت با GPT-4o-mini در مورد دیابت
- پایگاه داده غذایی: ۵۰+ غذای ایرانی با شاخص گلیسمیک
- اعلان‌های لحظه‌ای: WebSocket برای هشدارهای قند خون
- تجزیه‌وتحلیل پیشرفته: TIR، HbA1c، CV، میانگین روزانه
